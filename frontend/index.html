HCI TUIO Learning System

Interactive Learning with Physical Markers
Student Interface
Answer questions using physical TUIO markers on the table
Teacher Interface
Monitor students, configure tests, and view results

How to Use:

  1. Start ReactVision and point camera at TUIO markers
  2. Open Teacher interface to configure a test
  3. Open Student interface on the student's computer
  4. Student answers using physical markers on table
  5. System automatically scores and provides feedback
